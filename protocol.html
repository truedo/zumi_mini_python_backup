class protocol.Command_variable_byte

기반 클래스: ISerializable

classmethod getSize()
toArray()

size 값에 따라 동적으로 pack 포맷 문자열을 생성하여 데이터를 직렬화합니다. 첫 번째 바이트는 commandType, 나머지 바이트는 param1~param7 순서로 할당합니다.

classmethod parse(dataArray)

dataArray의 길이를 기준으로 동적으로 unpack 포맷 문자열을 생성하고, 해당 데이터를 Command_variable_byte 객체로 복원합니다. 입력된 size는 dataArray의 길이로 자동 할당합니다.
